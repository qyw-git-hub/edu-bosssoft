# wPanel 面板

常用的面板容器，搭配wPanelItem组件使用，建议注册到全局。

基础用法

自适应智付校园内容高度。

Attributes

参数说明类型可选值默认值
loading 面板加载 Boolean true / false false
disHeight 差值高 Number - 0
params 无需表格传递false，详细传参参照vxe官方文档，点击跳转 Object, Boolean - {}
pagination 无需页码传递false，详细传参参照element-ui页码官方文档，点击跳转 Object, Boolean - { page: 1, limit: 10, total: 0 }
tabbar 默认false，详细传参参照element-ui标签官方文档，点击跳转 Object, Boolean - { value: undefined, data: [], on: {} }
inline 当出现内外层都有w-panel，例如tab页签情况，在内层加上inline参数可适配 Boolean true / false false
leftTreeParams 是否增加左侧树内容 Object, Boolean - false
